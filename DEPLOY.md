# Go live checklist

Your site is now configured for GitHub Pages auto-deploy from `main`.

## 1) Commit and push

Run these commands from your project folder:

- git add .
- git commit -m "Set up GitHub Pages deployment"
- git push origin main

## 2) Enable Pages in GitHub (one-time)

In your GitHub repo:

- Open Settings
- Open Pages
- Under Build and deployment, choose Source: GitHub Actions

## 3) Live URL

After the workflow succeeds, your site will be available at:

https://jgillonline.github.io/jasdeep-landing-page/

Portfolio page direct link:

https://jgillonline.github.io/jasdeep-landing-page/portfolio.html

## 4) Optional custom domain

If you want your own domain:

- Add your domain in Settings > Pages
- Add matching DNS records at your domain provider
- (Optional) add a CNAME file at project root containing your domain only
